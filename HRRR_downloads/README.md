## HRRR Downloads

Brian Blaylock  
_Febuary 27, 2017_

A collection of download scripts for the MesoWest HRRR archive. These are run
daily after 6:00 PM Mountain Time and downloads the "previous UTC day."

### `HRRR_sequential_download.csh`

* download_hrrrX.py
* download_hrrrAK.py
